# Gemini CLI

Use `AGENTS.md` como fonte canonica deste repositorio.

## Instrucoes Especificas

- Ler `AGENTS.md` antes de analisar ou alterar arquivos.
- Usar `.agents/skills/` como fonte de verdade dos fluxos procedurais.
- Tratar `.gemini/commands/` como adaptadores finos para skills canonicas.
- Usar `@workspace.<command>` quando precisar acionar um wrapper da Gemini CLI.
- Carregar referencias adicionais apenas quando a tarefa exigir.
- Validar mudancas com comandos proporcionais ao risco.
