# Stack do Projeto

- **Tipo:** Toolkit Go reutilizável em monolito modular
- **Linguagem principal:** Go 1.26.0 no módulo
- **Padrão predominante:** package-by-component com ports/adapters localizados
- **HTTP:** go-chi e Fiber
- **Banco e persistência:** `database/sql`, pgx, otelsql, golang-migrate
- **Mensageria:** Kafka e RabbitMQ
- **Observabilidade:** OpenTelemetry, Prometheus, Jaeger, Grafana e Loki
- **Logging:** Zap
- **Testes:** Go test, Testify, Testcontainers
- **Qualidade:** golangci-lint, govulncheck, mockery
- **Infra local:** Docker Compose e Terraform
